# DCIT313-Group AI Squad-Smart-Academic-Advisor-Expert-System 

## Project Objective
The objective of this project is to design and implement a Knowledge-Based System (KBS) that functions as an Intelligent Agent[cite: 3]. [cite_start]This Smart Academic Advisor (SAA) is an AI-powered Expert System designed to assist students in making informed academic decisions, such as course planning and prerequisite management. ]The system demonstrates a clear mapping from perceptions (student data/career interests) to actions (course recommendations/alerts).

## Repository Structure
[cite_start]To comply with the mandatory architecture for DCIT 313, this repository is organized as follows[cite: 17, 18, 19]:

* [cite_start]**/knowledge_base**: Contains the `.pl` file (SWI-Prolog) housing the logical Facts and Rules that serve as the system's memory and intelligence[cite: 20].
* **/interface**: Contains the Python scripts using the `pyswip` library to run queries against the Prolog engine and manage user interaction[cite: 20].
* [cite_start]**/docs**: Contains documentation on Knowledge Engineering, including the sources used to acquire academic expertise[cite: 20].

---

## Technical Stack
* [cite_start]**Logic Engine**: SWI-Prolog for Symbolic AI and Inference[cite: 6].
* **User Interface**: Python with the `pyswip` library as the bridge to the Knowledge Base[cite: 7].
* [cite_start]**Version Control**: GitHub for collaboration and contribution tracking[cite: 8].

---

## System Functionality
[cite_start]The SAA Expert System is designed to solve common academic challenges[cite: 57, 66]:
* **Rule-Based Reasoning**: Uses IF-THEN logic to handle academic regulations, such as blocking registration if prerequisites are not met[cite: 78, 92, 94].
* [cite_start]**Decision Support**: Analyzes student records to recommend appropriate courses and identify academic risk levels[cite: 69, 71, 72].
* [cite_start]**Predictive Support**: Suggests improvement strategies for students on academic probation or with low GPAs[cite: 73, 90, 91].
